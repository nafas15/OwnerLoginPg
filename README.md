# AKSHA FARM - Employee Salary & Attendance App

A modern, premium-grade **Employee Salary & Attendance Management System** for AKSHA FARM. 

## Tech Stack
- **Frontend**: Single Page Application (HTML5, Vanilla ES6+ JS)
- **Styling**: Premium Vanilla CSS (Forest Green & Gold palette, glassmorphism, responsive)
- **Data Persistence**: Local-First Architecture utilizing `localStorage`
- **Dev Tooling**: Vite

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm run dev
   ```

## User Roles

### 👑 Owner (Admin)
- View summary dashboard (Total Employees, Payroll Expense, Daily Attendance status)
- Manage Employees (Add/Edit details)
- Mark Daily Attendance (Present, Absent, Half Day, Overtime)
- View Payroll & Process Salaries
- Generate and Print/Download Payslips
- Configurable settings (Company details, Overtime rates)

### 👷 Employee
- Login with Employee ID (e.g. `EMP001`)
- View monthly days worked, overtime hours, and attendance history
- View salary details
- View and download/print monthly Payslips
